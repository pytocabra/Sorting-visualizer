# Sorting-visualizer
This program is a visual implementation of three sorting algorithms:
- Selection sort
- Insertion sort
- Bubble sort

## Requiments
- Pygame
- Tkinter

Pygame is resposible for the main window in the project. Tkinter was used to create a small dialog window.
